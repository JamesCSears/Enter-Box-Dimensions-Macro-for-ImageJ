Enter Box Dimensions Macro:

The Enter Box Dimensions Macros was designed to make setting rectangular selections in ImageJ easier and more precise.  It is particularly useful for creating rectangular selections of the same size between different images taken at the same magnification.  


Installation:

First, put the Enter Box Dimensions Macro.txt file into the ImageJ macros file (for Mac: Applications/ImageJ/macros) To run the macro, have an image open in ImageJ or FIJI, then select Plugins>Macros>Run, then select Enter Box Dimensions Macro.txt.  You may also install it temporarily in the Macro list with Plugins>Macros>InstalI.  For frequent use you could input a keyboard shortcut in Plugins>Macro>Startup Macros… such as:


macro "Macro box [g]" { 

        runMacro("/Applications/ImageJ/macros/Enter Box Dimensions Macro.txt");

     } 


Using the Macro:

With an image open, run the macro.  In the Log screen the width, height, x location, and y location in pixels will be displayed of a rectangular selection or if no rectangular selection exists, the entire image.  A dialog box will appear to allow specific dimensions and x, y locations to be input.  Then, the macro will create a rectangular selection of those dimensions and location.  Note that using other selection tools will work, but information will be as if a rectangular selection was made, and the result will still be a rectangular selection.
